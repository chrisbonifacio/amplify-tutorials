# Getting Started with Amplify JS + Astro

```jsx
import React from 'react';
import {Amplify} from 'aws-amplify';
import awsconfig from './aws-exports';

Amplify.configure(awsconfig);

const App = () => {
  return <div>Hello World</div>;
};

export default App;
```
